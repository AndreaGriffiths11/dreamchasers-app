# Contributing to Dreamchasers 🎤

First off — thank you for being here. Whether you're fixing a typo, adding a resource, or suggesting a whole new track, you're helping someone learn to code. That matters.

## How to Contribute

1. **Fork** this repo
2. **Create a branch** (`git checkout -b my-fix`)
3. **Make your changes**
4. **Open a pull request** describing what you did and why

## What We're Looking For

- 🐛 **Bug fixes** — broken links, typos, unclear instructions
- 📚 **New resources** — found a great beginner tutorial? Add it to the "Go Deeper" section of the relevant track
- 💡 **New ideas** — have an idea for a new track or section? Open an issue first so we can discuss it
- 🌍 **Translations** — want to translate a track into another language? Let's talk

## Guidelines

- Keep the tone conversational and encouraging. This is for people who've never coded before.
- Don't assume prior knowledge. If something needs explaining, explain it.
- Test your links before submitting.

## Code of Conduct

Be kind. Be respectful. We're all here to learn.
