# 🎵 Track 0: The Pre-Game

> "Before the show starts, make sure your mic works."

`⏱️ 5 min · Pre-requisites`

## What You're About To Do

Quick check to make sure you're set up and ready to roll. No coding yet — just getting your stage ready.

## ✅ The Checklist

### 1. GitHub Account
- [ ] Go to [github.com](https://github.com) — are you signed in?
- [ ] If not, [sign up here](https://github.com/signup). It's free. No credit card needed.

### 2. Browser
- [ ] You're reading this, so ✅ you have one.

That's literally it. You don't need:
- ❌ A fancy computer
- ❌ Any software installed
- ❌ Prior coding experience
- ❌ A CS degree

### 3. Optional: Turn On GitHub Copilot (Free)
Copilot is an AI that helps you write code. You'll use it in Track 4, but you can turn it on now:

1. Go to [github.com/settings/copilot](https://github.com/settings/copilot)
2. Enable **GitHub Copilot**
3. Choose the free plan

It's like having a co-producer in the studio. 🎧

## 🧠 Quick Vocab

You don't need to memorize these — you'll learn them as you go. Just a heads-up so nothing sounds alien:

| Word | What It Means |
|------|--------------|
| **Repo** | A project folder on GitHub |
| **Commit** | A saved change (like a save point in a game) |
| **Branch** | A separate version of your project to experiment on |
| **Pull Request** | A way to propose changes and get feedback |
| **Fork** | Your own copy of someone else's project |

Everything's set. Let's make some noise.
