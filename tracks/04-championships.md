## What You're About To Do

Use AI to help you code. GitHub Copilot is like having a producer in the studio — it suggests code, explains things, and helps you build faster.

## Copilot in the Browser

You don't need to install anything. Copilot works right inside GitHub.

### Try It: Ask Copilot a Question
1. Go to [github.com/copilot](https://github.com/copilot)
2. Type: "How do I make a website with HTML?"
3. Copilot will write the code for you and explain it

That's it. You just used AI to code. 🤯

![Copilot Chat](assets/screenshots/04-copilot-chat.png)

### Try It: Copilot in Your Repo
1. Open any file in your repo
2. Click the **Edit** pencil ✏️
3. Start typing — Copilot suggests completions in gray
4. Press **Tab** to accept

## Copilot CLI (Terminal)

If you want to go deeper, Copilot CLI lets you code with AI in your terminal.

```bash
# Install it
curl -fsSL https://gh.io/copilot-install | bash

# Start it
copilot
```

It works with Claude, GPT, and Gemini. You pick the model. All free with your GitHub account.

Tell it what you want in plain English:
> "Build me a website that shows my favorite songs"

And it builds it. For real.
